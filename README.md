# RepliFa Portfolio Website

This is a static, deployable one-page portfolio website built from the uploaded RepliFa lookbook.

## Files
- `index.html` — main page
- `styles.css` — visual system and responsive layout
- `script.js` — filters, modal gallery, mobile nav, reveal animation
- `assets/` — rendered images from the lookbook

## How to preview
Open `index.html` in a browser.

## How to publish
You can upload the entire folder to Netlify, Vercel, GitHub Pages, or your existing hosting for lenyitaima.com.

## Privacy note
The public contact section intentionally omits phone numbers. Add them back in `index.html` only if you want them public.
